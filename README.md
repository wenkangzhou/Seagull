# 「海鸥」，一个安卓平台的HybridApp
##SeagullClient
安卓客户端代码。主要就是搭了一个安卓的壳子，实现了Webview以及Native与Js交互的接口。
##SeagullH5
Hybrid/H5代码。整体布局用了「iscroll」，来实现横、纵的滑动效果；样式参考「一个」；音乐播放器用了「jPlayer」；用Localstorage缓存数据；用manifest配置文件实现离线应用。
##SeagullServer
后台和接口代码。使用了Java的SSH框架。
